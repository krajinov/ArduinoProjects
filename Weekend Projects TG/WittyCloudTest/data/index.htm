ESP8266 Witty Cloud Board Demo

This is a sample sketch for testing all the peripherals on the ESP8266 Witty Cloud board. It also demonstrates some notable features of the ESP8266/Arduino port like hosting a webserver and a websockets server. This project is heavily inspired by the ColorChord: Embeddedproject.
